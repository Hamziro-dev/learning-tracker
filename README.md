# Learning Tracker 📊  
Python × Kivyで作る学習進捗管理アプリ  

## 🎯 概要
学習項目と達成率を可視化するGUIアプリ。  
Python × Kivyで構築し、ローカル環境で動作する。

## ⚙️ 技術仕様
| 項目 | 内容 |
|------|------|
| フレームワーク | Kivy, KivyMD |
| データベース | SQLite（予定） |
| 構成パターン | MVC風分離構造（ui / logic / data） |
| 主な機能 | 学習記録の登録・削除・集計・可視化 |
| 目標 | GUI + データ処理 + 永続化の統合学習 |

## 💡 機能（予定含む）
- 学習項目の登録・編集・削除  
- 日付ごとの進捗記録  
- グラフ表示（週・月単位）  
- CSVエクスポート  

## 🧠 開発目的
業務で必要な UI/状態管理/永続化/テストの基礎を一つの成果物で同時に鍛えるため、Python × Kivy で Learning Tracker を開発。
これにより （1）学習ログの記録・可視化 と （2）GUI・DB・テストの実装経験 を同時に積み上げる。

また、継続的な学習トラッキングを支援するツールを作る。
